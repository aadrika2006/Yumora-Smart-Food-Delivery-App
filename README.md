🍔 Yumora – Smart Food Delivery App
Yumora is a modern Android-based food delivery application built using Java in Android Studio. The app allows users to browse food items, explore restaurants, and place orders seamlessly with a clean and interactive user interface.

Inspired by platforms like Swiggy and Zomato, Yumora introduces smart and user-friendly features such as micro-ordering (half portions), voice-based ordering, local home chef discovery, and price drop alerts.

The system is designed to provide a fast, efficient, and personalized food ordering experience while also optimizing delivery processes using search-based logic.

✨ Features
🍽 Browse restaurants and food items
🔍 Smart search functionality
🛒 Add to cart & place orders
🎤 Voice-based food ordering
🍱 Micro-order (half/full portion)
🏠 Local home chef integration
🔔 Price drop alerts
📦 Order tracking system

🛠 Tech Stack
Frontend: Android Studio (Java + XML)
Backend: Firebase / Spring Boot
Database: Firestore / MySQL
APIs: REST APIs
