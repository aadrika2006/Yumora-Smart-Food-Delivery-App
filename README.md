# 🍔 Yumora – Smart Food Delivery App

> “Order smart. Eat better.”

Yumora is a modern Android-based food delivery application built using **Java in Android Studio**. The app provides a seamless and interactive food ordering experience with innovative features like micro-ordering, voice-based ordering, local home chef discovery, and price drop alerts.

Inspired by platforms like Swiggy and Zomato, Yumora focuses on **speed, usability, and smart decision-making** while maintaining a clean and visually appealing UI.

---

## 📱 App Overview

Yumora allows users to:
- Browse food items and restaurants
- Search and filter dishes
- Add items to cart and place orders
- Track delivery status
- Discover local home chefs

The system also incorporates smart features to improve the user experience and optimize ordering.

---

## ✨ Features

### 🍽 Core Features
- Browse restaurants and food items
- Search functionality
- Add to cart & order placement
- Order tracking system

### 🚀 Smart Features
- 🎤 **Voice Ordering** – Order food using voice commands  
- 🍱 **Micro-Order Mode** – Choose half/full portion  
- 🏠 **Home Chef Mode** – Discover homemade food  
- 🔔 **Price Drop Alerts** – Get notified for discounts  
- 📊 **Meal Planning (Optional)** – Budget-based suggestions
